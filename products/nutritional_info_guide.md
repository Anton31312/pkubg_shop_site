# Руководство по заполнению пищевой ценности продуктов

## Структура данных nutritional_info

Поле `nutritional_info` содержит подробную информацию о пищевой ценности продукта в формате JSON.

### Основные разделы:

#### 1. per_100g - Пищевая ценность на 100г
```json
"per_100g": {
    "calories": 250,        // Калорийность в ккал
    "proteins": 8.5,        // Белки в граммах
    "fats": 3.2,           // Жиры в граммах
    "carbohydrates": 45.0,  // Углеводы в граммах
    "fiber": 5.5,          // Клетчатка в граммах
    "sugar": 2.1,          // Сахар в граммах
    "salt": 1.2,           // Соль в граммах
    "sodium": 480.0        // Натрий в миллиграммах
}
```

#### 2. vitamins - Витамины
```json
"vitamins": {
    "vitamin_a": 15.0,      // Витамин A в микрограммах
    "vitamin_c": 0.5,       // Витамин C в миллиграммах
    "vitamin_d": 0.0,       // Витамин D в микрограммах
    "vitamin_e": 1.2,       // Витамин E в миллиграммах
    "vitamin_k": 2.5,       // Витамин K в микрограммах
    "thiamine_b1": 0.15,    // Тиамин (B1) в миллиграммах
    "riboflavin_b2": 0.08,  // Рибофлавин (B2) в миллиграммах
    "niacin_b3": 2.1,       // Ниацин (B3) в миллиграммах
    "vitamin_b6": 0.12,     // Витамин B6 в миллиграммах
    "folate_b9": 25.0,      // Фолат (B9) в микрограммах
    "vitamin_b12": 0.0      // Витамин B12 в микрограммах
}
```

#### 3. minerals - Минералы
```json
"minerals": {
    "calcium": 120.0,       // Кальций в миллиграммах
    "iron": 2.5,           // Железо в миллиграммах
    "magnesium": 45.0,     // Магний в миллиграммах
    "phosphorus": 180.0,   // Фосфор в миллиграммах
    "potassium": 250.0,    // Калий в миллиграммах
    "zinc": 1.8,           // Цинк в миллиграммах
    "copper": 0.3,         // Медь в миллиграммах
    "manganese": 0.8,      // Марганец в миллиграммах
    "selenium": 12.0       // Селен в микрограммах
}
```

#### 4. allergens - Аллергены
```json
"allergens": ["глютен", "молоко", "яйца"]
```

Возможные значения:
- "глютен"
- "молоко" 
- "яйца"
- "орехи"
- "арахис"
- "соя"
- "рыба"
- "моллюски"
- "кунжут"
- "горчица"
- "сельдерей"
- "люпин"
- "диоксид серы"

#### 5. dietary_info - Диетическая информация
```json
"dietary_info": {
    "is_vegetarian": true,   // Подходит для вегетарианцев
    "is_vegan": false,       // Подходит для веганов
    "is_gluten_free": true,  // Без глютена
    "is_lactose_free": false, // Без лактозы
    "is_sugar_free": false,  // Без сахара
    "is_organic": true,      // Органический продукт
    "is_kosher": false,      // Кошерный
    "is_halal": false        // Халяльный
}
```

#### 6. storage_info - Информация о хранении
```json
"storage_info": {
    "temperature": "комнатная температура",  // "холодильник", "морозилка", "комнатная температура"
    "shelf_life_days": 7,                    // Срок годности в днях
    "storage_conditions": "Хранить в сухом прохладном месте, избегать прямых солнечных лучей"
}
```

#### 7. serving_info - Информация о порциях
```json
"serving_info": {
    "serving_size_g": 50,                    // Размер порции в граммах
    "servings_per_package": 10,              // Количество порций в упаковке
    "preparation_instructions": "Разогреть в духовке при 180°C в течение 5-7 минут"
}
```

## Примеры заполнения для разных типов продуктов

### Хлеб безглютеновый
- Калории: 250-280 ккал/100г
- Белки: 6-10г
- Жиры: 2-5г
- Углеводы: 45-55г
- Клетчатка: 4-8г

### Молочные продукты
- Обязательно указать аллерген "молоко"
- is_lactose_free: false (если не безлактозный)
- is_vegan: false

### Мясные продукты
- Высокое содержание белка (15-25г)
- is_vegetarian: false
- is_vegan: false

## Важные замечания

1. **Все значения указываются на 100г продукта**
2. **Не удаляйте структуру JSON** - только изменяйте значения
3. **Используйте точку как разделитель десятичных дробей** (не запятую)
4. **Проверяйте данные** по официальным источникам или лабораторным анализам
5. **Обязательно указывайте аллергены** согласно законодательству
